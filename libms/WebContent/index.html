1 Overview

LIBMS(Library Management System)后台采用Java语言开发；控制层采用纯Servlet返回JSON,使用jackson包来转换Java对象为JSON字符串；数据访问层使用JDBC，数据库采用H2内存数据库。

2 数据库设计

数据库名为test, 账号是sa, 密码为空。

2.1 用户表(user)

 CREATE TABLE USER(
        uid int NOT NULL AUTO_INCREMENT ,
        uname VARCHAR(50),
        upwd VARCHAR(20),
        PRIMARY KEY(uid)
)

2.2 图书类别表 （booktype）

CREATE TABLE BOOKTYPE(
        books int NOT NULL AUTO_INCREMENT ,
        tit VARCHAR(500),
        bigImg VARCHAR(100),
        PRIMARY KEY(books)
)

2.3 图书表(book)

CREATE TABLE BOOKCOMMENT(
        id int NOT NULL AUTO_INCREMENT ,
        tit VARCHAR(100),
        intro VARCHAR(200),
        words text,
        author VARCHAR(50),
        aPhoto VARCHAR(100),
        bigImg VARCHAR(100),
        clickNum int,
        great int,
        PRIMARY KEY(id)
)

3 接口说明

3.1 用户模块

URL路径：http://localhost:8080/user

参数说明:

  • 初始化数据：dispatch=init 会删除user表并重新创建表，并添加初始数据如下：

    完整路径：http://localhost:8080/user?dispatch=init 

      [{
        "userId": 1,
        "userName": "root",
        "userPwd": "123456"
}, {
        "userId": 2,
        "userName": "guest",
        "userPwd": "123456"
}]
  • 登录：dispatch=login&userName=root&userPwd=123456

完整路径：http://localhost:8080/user?dispatch=login&userName=root&userPwd=123456

登录验证，需要提交两个参数userName和userPwd。根据userName和userPwd从user表查询数据，如果存在就登录成功，把用户存入session, 属性名叫user，并返回json对象 

{"result":"1"}

如果没有查询到对应的用户，则登录失败，返回Json对象 

{"result":"0"}
  • 退出登录：dispatch=logout

完整路径：http://localhost:8080/user?dispatch=logout

退出登录会清除session,返回字符串logout

  • 不带参数执行Init初始化数据操作，返回default字符串

完整路径： http://localhost:8080/user

  • 新增图书评论： dispatch=add 同时需要提交如下表单数据：
    • aPhoto
    • author
    • bigImg
    • clickNum
    • great
    • intro
    • tit
    • words

执行完成后会在bookcomment表添加一条记录，返回字符串add

  • 删除图书评论： dispatch=delete 需要提交参数: id

    执行完成后会删除bookcomment表中相应Id的记录，返回字符串delete

  • 修改图书评论： dispatch=update 同时需要提交如下表单数据：
    • aPhoto
    • author
    • bigImg
    • clickNum
    • great
    • intro
    • tit
    • words
    • id

执行完成后会修改bookcomment表相应id的数据

  • 查询全表数据：dispatch=list

执行全表查询，并返回json数据

  • 查询指定id的数据： dispatch=getbyid

同时需要提交参数id 执行完成后返回一条记录的json数据
